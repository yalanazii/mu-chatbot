# MU-chatbot
A helpful AI-powered chatbot that Marymount University students can utilize to quickly access important information, receive accurate answers to common questions, and navigate campus resources without needing to search through multiple webpages, simplifying the student experience by offering instant support for academic deadlines, financial dates, campus events, library hours, and department contacts. 
- This chatbot currently only uses information for the 2026 Spring semester.

## You may ask it questions such as:
### When does spring break start?
### When do I need to pay tuition?
### When is Easter break?
### Any activities this week?
### Is the library open?
### Financial aid contact?
